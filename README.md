# jaxus
Ultrasound beamforming and processing in JAX.

## Installation
```bash
pip install .
```
or if you want to install in editable mode:
```bash
pip install -e .
```

The project also depends on JAX. You can install it via pip:
```bash
pip install jax[cpu]
```
or
```bash
pip install jax[cuda122]
```
